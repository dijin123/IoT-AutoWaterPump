# IoT-AutoWaterPump
Auto On/Off Water Pump based on Water Level in the Tank

## Componet List
* NODEMCU - ESP8266 Wifi Dev Board on CP2102
* Ultrasonic Sensor Module - HC-SR04
* Tiny Real Time Clock - DS107
